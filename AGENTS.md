# Agent Instructions

This repository contains configuration for automated agents.

## Agent skills

### Issue tracker

Issues are tracked on GitHub using the `gh` CLI. Pull requests are not treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Using default canonical triage labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

The repository is configured as a single-context repository. See `docs/agents/domain.md`.
